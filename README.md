Цель работы: приобрести практические навыки ручного тестирования
методом "белого ящика".
Описание:WPF приложение для вычисления математических функций.
Функциональность:
Страница 1
Вычисление кусочной функции:
y =
sin(5f(x)+3m|f(x)|), если -1 < m < x  
cos(3f(x)+5m|f(x)|), если x > m  
(f(x)+m)^2, если x = m  
где f(x) выбирается:
- sh(x)
- x^2
- e^x
Страница 2
Вычисление функции:
l =
2f(x)^3 + 3p^2, если x > |p|  
|f(x) - p|, если 3 < x < |p|  
(f(x) - p)^2, если x = |p|
Страница 3
Построение графика функции:
y = 0.0025bx^3 + √x + e^0.82
